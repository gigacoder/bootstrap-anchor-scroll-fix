Fix the scrolling issue with CSS

In style.css add folloring properties padding-top and margin-top in the line of the targeted id. For example. In this Template we will add those properties in the class featurette.

SERVICES

Have a look At the stylesheet. 

                .featurette {
                    padding-left:15px;
                    padding-right: 15px;
                    overflow: hidden;
                    padding-top: 60px;
                    margin-top:-60px;
                }
